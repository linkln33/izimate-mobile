# Professional Notification Sounds

These sounds were downloaded from notificationsounds.com and are licensed under Creative Commons Attribution.

## Files

- **notification.wav** - Slick (short, subtle notification)
- **booking.wav** - Joyous (melodic chime for bookings)
- **reminder.wav** - Dutifully (polite attention-grabber)
- **message.wav** - Pristine (clear, pleasant message tone)
- **alarm.wav** - Sharp (cuts through noise for urgent alarms)

## Conversion Status

Currently, the professional sounds are in MP3 format (`*_temp.mp3`). To use them:

1. Install ffmpeg: `brew install ffmpeg`
2. Run: `node scripts/convert-mp3-to-wav.js`
3. Or convert manually using an online converter
